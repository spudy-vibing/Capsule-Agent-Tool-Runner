Core Features

Everything you need for secure, auditable agent execution

🔒

Deny-by-Default

Everything is blocked unless explicitly allowed. Policies define precise boundaries for file, network, and shell operations.

📜

Full Audit Logging

All tool calls and results stored in SQLite with cryptographic hashes. Complete traceability for every operation.

🔁

Deterministic Replay

Reproduce exact behavior from past executions. Debug issues and verify results with confidence.

📈

Rich Reporting

Timeline views and JSON export for analysis. Understand what happened at every step.

🔌

Extensible Tools

Clean plugin architecture for adding new tools. Built-in support for fs, shell, and HTTP operations.

🤖

Dynamic Agent Mode

Let a local LLM decide tool calls dynamically. Supports Ollama with multiple model options.

Quick Start

Up and running in 5 minutes

Installation
# Clone and install
$ git clone https://github.com/spudy-vibing/Capsule---Agent-tool-runner.git
$ cd capsule
$ pip install -e .

# Run a plan with policy enforcement
$ capsule run plan.yaml --policy policy.yaml

# Or use dynamic agent mode with local LLM
$ capsule agent run "List all Python files in src/" \
    --policy policy.yaml \
    --model qwen2.5:7b \
    --pretty
